# HubWale

## Project Requirement

- PHP : 7.3 Or Greater,
- Node : 14 Or Greater
- NPM : 8.5 Or Greater
- MySQL : 8.0 Or Greater

## Installation

Install the dependencies and start the server.

##### Clone repositery

```sh
git clone https://github.com/aakashdesai-pro/sonne-erp.git
```

##### Go to project directory
```sh
cd sonne-erp
```

##### Install PHP dependencies
```sh
composer install
```

##### Install Javacript dependencies
```sh
npm install
```

##### Run project
```sh
php artisan serve
```
